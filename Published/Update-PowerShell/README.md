# Update-PowerShell

Updates PowerShell to Version 5.1

## Syntax
```PowerShell
Update-PowerShell.ps1
```
## Description

Updates a target machine to PowerShell V5.1 if needed.

## Examples


###  Example 1 
```PowerShell
Update-PowerShell.ps1
```
